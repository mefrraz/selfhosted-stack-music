The Stack

Core music server — Subsonic API, library indexing, themes, multi-library (v0.58+), plugins. Your personal Spotify.

Features

Everything for a production-grade personal music server — nothing you don't need.

🎧

Stream Everywhere

Subsonic API works with Feishin, Substreamer, Navic, Amperfy, and dozens more — on every platform.

📥

Automated Acquisition

Deemix downloads FLAC or MP3 with perfect metadata. Spotify public playlist fallback included in v3.13.x.

Synced Lyrics

Dracula, Nord, AMusic, SquiddiesGlass and more. Plugin system introduced in latest Navidrome releases.

The ARL cookie authenticates Deemix with your Deezer account. It's required before starting the stack.

O cookie ARL autentica o Deemix com a tua conta Deezer. É obrigatório antes de iniciar a stack.

  • Login to Deezer on a desktop browser
  • Faz login no Deezer num browser desktop
  • Open DevTools: F12 → Application → Cookies → deezer.com
  • Abre DevTools: F12 → Application → Cookies → deezer.com
  • Copy the value of the arl cookie and paste it into Deemix settings
  • Copia o valor do cookie arl e cola-o nas definições do Deemix

🐳 Docker Compose

Minimal setup to get Navidrome and Deemix running:

Setup mínimo para correr o Navidrome e o Deemix:

# docker-compose.yml
services:
  navidrome:
    image: deluan/navidrome:latest
    ports: ["4533:4533"]
    volumes:
      - ./data/navidrome:/data
      - ./music:/music:ro
    environment:
      ND_LOGLEVEL: info
      ND_UI_VAMP_THEME: dark

  deemix:
    image: ghcr.io/bambanah/deemix:latest
    ports: ["6595:6595"]
    volumes:
      - ./music:/downloads
      - ./data/deemix:/config
    environment:
      PUID: 1000
      PGID: 1000
      UMASK_SET: 022
# Start the stack
docker compose up -d

🟣 Navidrome

Open-source music server written in Go — your personal Spotify. Handles library indexing and exposes the Subsonic API.

Servidor de música open-source escrito em Go — o teu Spotify pessoal. Gere a indexação da biblioteca e expõe a API Subsonic.

Notable features (v0.58+): Multi-library support, Scrobble History, Instant Mix via Last.fm or Deezer, plugin system.

Funcionalidades notáveis (v0.58+): Suporte multi-biblioteca, Scrobble History, Instant Mix via Last.fm ou Deezer, sistema de plugins.

Recommended clients:

Clientes recomendados:

  • Feishin — Desktop (Windows / macOS / Linux) · React + ElectronDesktop (Windows / macOS / Linux) · React + Electron
  • Substreamer — iOS & Android
  • Navic — Modern iOS/Android client · zero telemetryCliente moderno iOS/Android · zero telemetria

🔵 Deemix (Bambanah)

Actively maintained Docker-optimized fork of Deemix. Downloads FLAC or MP3 with complete ID3v2 metadata. Current version: 3.13.6.

Fork do Deemix ativamente mantido e otimizado para Docker. Descarrega FLAC ou MP3 com metadados ID3v2 completos. Versão atual: 3.13.6.

⚠️  Disable the built-in lyrics download — it's unstable. Use LRCGET for reliable lyric embedding.Desativa o download de letras nativo — é instável. Usa o LRCGET para embutir letras de forma fiável.

Spotify fallback (v3.13.x): When a public Spotify playlist returns 404, Deemix now parses track IDs from the public page and resolves them individually.

Fallback Spotify (v3.13.x): Quando uma playlist pública do Spotify devolve 404, o Deemix agora analisa os IDs da página pública e resolve-os individualmente.

# Pull standalone
docker run -d --name Deemix \
  -v /path/to/music:/downloads \
  -v /path/to/config:/config \
  -p 6595:6595 \
  ghcr.io/bambanah/deemix:latest

🟢 LRCGET

The official desktop client for LRCLIB — the largest open, community-built lyrics database. Scans your library and embeds synchronized LRC lyrics into audio tags.

O cliente oficial do LRCLIB — a maior base de dados de letras open-source. Analisa a biblioteca e embute letras LRC sincronizadas nas tags dos ficheiros de áudio.

  • Set mode to Embed lyrics into audio files (ID3v2) — not external .lrc
  • Define o modo para Embutir letras nos ficheiros de áudio (ID3v2) — não .lrc externos
  • Supports custom self-hosted LRCLIB instances
  • Suporta instâncias LRCLIB personalizadas (self-hosted)
  • Cross-platform: Windows, macOS, Linux — written in Rust
  • Multi-plataforma: Windows, macOS, Linux — escrito em Rust
  • Navidrome detects embedded lyrics automatically — zero extra config
  • O Navidrome deteta letras embutidas automaticamente — sem configuração extra

O navidrome-radio.py usa a API Radio-Browser para injetar estações de rádio diretamente na base de dados SQLite navidrome.db — sem reinício do contentor.

  • Search by name (e.g. Comercial, RFM), genre, or country
  • Pesquisa por nome (ex: Comercial, RFM), género ou país
  • Pagination support for large result sets
  • Suporte a paginação para resultados extensos
  • Duplicate detection — stations are never added twice
  • Deteção de duplicados — estações nunca são adicionadas duas vezes
